Removed: Feature Strings

This appendix is no longer part of the SVG specification.

No new feature strings are added in SVG 2, as the requiredFeatures attribute has been dropped. Authors wishing to use feature testing to exclude older SVG viewers should use the strings as defined in SVG 1.1.

You can view the appendix in SVG 1.1, or go back to the main page for SVG 2.
